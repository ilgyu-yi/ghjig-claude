- Fixed the documented Directive **unblock** path being a dead end: a Blocked Directive is Active (`directive` + `status:blocked`, no `status:proposed`), so `/activate <N>` errored "already Active" before reaching the reviewer, and even on a pass it only removed `status:proposed` — never `status:blocked`. `/activate` is now blocked-aware: it treats `status:blocked` as a re-activatable precondition, re-runs `activation-reviewer`, and on `pass` removes whichever of `status:proposed`/`status:blocked`/`awaiting-author` are present (so the mirror reconciles Status). Batch mode still does not auto-unblock (single-mode `/activate <N>` only). (#215)
