- Broadened the staged-secret scanner's OpenAI key pattern to catch current-generation key formats (`sk-proj-…`, `sk-svcacct-…`, `sk-admin-…`, which contain `-`/`_`). The old `sk-[A-Za-z0-9]{40,}` stopped at the first `-` after the prefix, so the length floor was never met and modern keys passed the scan as a false negative. The `{40,}` floor is retained to limit false positives on short `sk-` identifiers. (#214)
